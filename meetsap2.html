ABOUT THE LOGO

Since SAP2 operates similarly to a microprocessor,
the logo was made after its image. The microprocessor,
most or all of the functions of a computer's central
processing unit(CPU) incorporated in a single integrated
circuit. Aside from this technical basis, the details
of our logo bears symbolism of our section. The four
corners of the diamond represents stability. This shows
that even if we face tough moments, we are still able
to withstand the pressure that we engineering students
confront. The separation between the two diamonds depicts
how indifferent we are to each other but along the way
we adapt together because we figured out that our bond
of friendship is more important than any of our flaws.

Program/Design Mode

#1

This button returns the page to the start menu.

#2

This field contains list of intructions and op-codes.

#3

This displays the list of addresses.

#4

This displays the list of contents.

#5

This displays the list of mnemonics.

#6

The address field displays the address encoded by the user with the help of the HEX keyboard.

#7

The content field displays the content encoded by the user using the HEX keyboard.

#8

HEX keyboard has 18 keys. It consists of numbers 0-9 and letters A-F.
The other remaining keys are delete key and enter key.

#9

This button converts Decimal to Hexadecimal.

#10

This button clears or deletes the contents of the list box.

#11

This button writes the contents and addresses for run mode.

#12

The label indicates whether you are in the program mode or run mode.
If the label is highlighted blue, you are currently using the specified page.

Run Mode

#1. Back

This button returns the page to the program mode window.

#2. Address

This contains the address of your contents.

#3. Contents

This contains the contents you write during program mode.

#4. Block

This is the block diagram of SAP-2. It tells you the
value of every block during the T-states. And at the end
of the program, the value will be displayed in Output port in Hex form.

#5. Clear

Return the program to fetch cycle (first part of clocking)

#6. Clock

Starts clocking

#7. Auto/Manual Mode

Option that tells if you want the program to run
in automatic mode or manual mode.

#8. Label

The label indicates whether you are in the program mode or run mode.
If the label is highlighted blue, you are currently using the specified page.
